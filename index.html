BeatVortex: Beat Saber Support for Vortex

Documentation Install in Vortex Nexus Mods

Adds support for Beat Saber mods and maps to the Vortex mod manager. Fully open-source.

BeatVortex is a simple extension to add Beat Saber support to the Vortex mod manager. With this extension you can install, remove and manage your Beat Saber mods and custom songs using Vortex, perfect for players already familiar with Vortex.

This project is unofficial! We have no association with Nexus Mods nor with Beat Games and is only a community project.

Full Vortex integration

Manage your Beat Saber mods the same way you manage any other Vortex-supported game.

Contributions welcome!

BeatVortex is fully open-source and open to community contributions and fixes!
