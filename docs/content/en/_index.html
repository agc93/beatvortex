{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} BeatVortex is a simple extension to add Beat Saber support to the Vortex mod manager. With this extension you can install, remove and manage your Beat Saber mods and custom songs using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with Beat Games and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your Beat Saber mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/beatvortex" %}} BeatVortex is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="OneClick Install" %}} BeatVortex includes alpha support for one-click installs from BeatSaver for easier song downloads. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
